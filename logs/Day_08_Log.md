## What I Learned

* Two Sum on platforms usually asks for indexes, not values.
* Stock buy/sell requires tracking the minimum price before calculating profit.
* Kadane’s Algorithm is important for maximum subarray problems.
* Move Zeroes requires modifying the array in-place while keeping non-zero order.
* Remove duplicates from sorted array uses a two-pointer style approach.
* Single Number can be solved using XOR because duplicate numbers cancel out.
* Contains Duplicate can be solved efficiently using set.
* Plus One requires careful handling of carry, especially when all digits are 9.
* Mixed aptitude revision helps maintain speed and accuracy across different topics.
