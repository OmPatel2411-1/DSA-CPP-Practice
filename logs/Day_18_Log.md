## What I Learned

* Hash sets are useful when fast existence checking is needed.
* Hash maps are useful when counting frequency.
* Jewels and Stones uses set lookup.
* Pangram checking can be done by storing all unique letters.
* Consistent Strings checks whether every character belongs to the allowed set.
* Unique Number of Occurrences requires checking uniqueness of frequency values.
* Sum of Unique Elements requires counting first and adding only frequency-one elements.
* Similar hash set/map logic can apply to both strings and arrays.
