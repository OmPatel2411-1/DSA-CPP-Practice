## What I Learned

* Palindrome problems usually use two-pointer logic.
* Valid Palindrome requires cleaning or skipping non-alphanumeric characters.
* `tolower()` helps compare characters without case issues.
* Valid Palindrome II requires trying both options after one mismatch.
* Is Subsequence checks whether all characters of one string appear in order inside another string.
* Reverse Vowels uses two pointers but only swaps vowels.
* Length of Last Word needs careful handling of trailing spaces.
* String problems become easier when the condition is separated clearly before writing code.
